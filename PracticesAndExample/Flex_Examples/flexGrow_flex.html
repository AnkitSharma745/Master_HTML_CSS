flex-grow vs flex

1. flex-grow (Space Distribution)

The flex-grow property allows an item to grow relative to other items in the container. In this example, the first item grows 2 times more than the second item.

Item 1 (flex-grow: 2)
Item 2 (flex-grow: 1)

2. flex (Shorthand for flex-grow, flex-shrink, and flex-basis)

The flex property is a shorthand for flex-grow, flex-shrink, and flex-basis. It allows you to set all these properties in one declaration.

Item 1 (flex: 1)
Item 2 (flex: 2 1 150px)

Key Takeaways:

  • flex-grow: Controls how much an item should grow in relation to others when there's extra space in the container. The higher the value, the more space it takes up.
  • flex: A shorthand that sets all three flex properties (grow, shrink, basis) at once. Use it when you want to quickly set multiple properties in one line.
