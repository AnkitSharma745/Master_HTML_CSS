flex-shrink vs flex

1. flex-shrink (Controlling Shrinking)

The flex-shrink property defines the ability of an item to shrink when there's not enough space in the container. In this example, the first item can shrink, while the second item won't shrink at all.

Item 1 (flex-shrink: 1)
Item 2 (flex-shrink: 0)

2. flex (Shorthand for flex-grow, flex-shrink, and flex-basis)

The flex property is a shorthand for flex-grow, flex-shrink, and flex-basis. It allows you to set all these properties in one declaration.

Item 1 (flex: 1 1 100px)
Item 2 (flex: 1 0 150px)

Key Takeaways:

  • flex-shrink: Controls the shrinking behavior of an item when there’s not enough space in the container. A value of 0 means the item will not shrink.
  • flex: A shorthand property that sets flex-grow, flex-shrink, and flex-basis in one declaration. It includes shrink behavior along with grow and basis values.

Best Practice: Use flex-shrink when you want to control how an individual item shrinks when there’s not enough space. Use flex when you want to set all space-related properties (grow, shrink, basis) in one go.
