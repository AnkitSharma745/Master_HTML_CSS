1. Default Flex-Shrink (flex-shrink: 1)

By default, all flex items have a flex-shrink value of 1, meaning they shrink equally when space is limited.

2. Custom Flex-Shrink

The flex-shrink property determines how much an item shrinks relative to others. Items with higher values shrink more, while items with a value of 0 do not shrink at all.

Key Takeaways for flex-shrink:

  • Default: The default value is 1, meaning all items shrink equally.
  • Custom Shrinkage: Items with higher flex-shrink values shrink more.
  • No Shrinkage: Items with flex-shrink: 0 do not shrink.

Best Practice: Use flex-shrink to control how items adjust when space is constrained.
